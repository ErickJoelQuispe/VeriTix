# Markdown puro

Este directorio contiene documentación en Markdown sin tooling de compilación.

- `ARCHITECHTURE.md`: arquitectura y boundaries del sistema.
- `STATUS.md`: snapshot de implementación, gaps y desalineaciones.
- `REQUIREMENTS.md`: trazabilidad RF implementado / pendiente.
- `design.md`: contrato para componentes de página.
- `tfg.md`: fuente base de alcance y requerimientos.
