# Diapositivas visuales — VeriTix Slidev

Esta carpeta contiene el contenido **visual** de cada diapositiva: lo que la audiencia ve en pantalla.

## Propósito

Cada archivo `.md` es una diapositiva incluida desde `slides.md` mediante la directiva `src:` de Slidev. Contienen solo el contenido que se renderiza: títulos, diagramas, iconos y viñetas mínimas.

Los guiones orales del presentador están en `docs/slidev/notes/`.

## Convenciones

- Contenido visual únicamente (headlines, bullets, placeholders, grids).
- Sin párrafos extensos ni texto narrativo.
- Estilo oscuro consistente (fondo `#0d0d1a`).
- Demo es un separador, no parte del cuerpo de la presentación.
