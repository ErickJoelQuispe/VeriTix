# Guion — Sostenibilidad

## Contexto para el presentador

Esta diapositiva vincula el proyecto con los Objetivos de Desarrollo Sostenible. Es importante para demostrar que el proyecto tiene impacto más allá de lo técnico.

## Qué decir"

"VeriTix contribuye a tres ODS.

ODS 9 — Industria e innovación: el sistema está construido con arquitectura modular, capas bien separadas y API documentada. Esto facilita que otros desarrolladores se integren y que el proyecto evolucione.

ODS 12 — Consumo responsable: el código es eficiente, usa caché con Redis y consultas optimizadas. El frontend es ligero. Y al digitalizar procesos que antes requerían papel y desplazamientos físicos, reducimos el consumo de recursos.

ODS 13 — Acción por el clima: reemplazamos procesos manuales que dependían de papel, llamadas telefónicas y coordinación presencial por un sistema digital centralizado. Los tickets digitales eliminan la necesidad de imprimir entradas físicas.

En resumen, VeriTix digitaliza un flujo que históricamente dependía de papel y procesos manuales, reduciendo la huella operativa y aumentando la trazabilidad."
