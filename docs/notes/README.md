# Notas del presentador — VeriTix

Esta carpeta contiene los guiones orales para cada diapositiva de la presentación Slidev.

## Propósito

Estos archivos **no se renderizan** en la presentación. Son exclusivamente para el uso del presentador durante la exposición ante tribunal o audiencia ejecutiva.

Cada archivo incluye:
- Contexto de la diapositiva
- Texto sugerido para decir en voz alta
- Tiempo estimado de exposición

## Relación con src/

El contenido visual que ve la audiencia está en `docs/slidev/src/`. Estos guiones son el complemento oral: lo que se dice mientras se muestra cada diapositiva.
