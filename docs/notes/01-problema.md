# Guion — Problema del sector

## Contexto para el presentador

Esta diapositiva presenta el problema que VeriTix resuelve. Es la primera diapositiva de contenido después de la portada.

## Qué decir

"El sector de organización de eventos se enfrenta a tres problemas principales.

Los organizadores hoy usan herramientas desconectadas: una para vender entradas, otra para controlar el aforo, otra para gestionar el personal. No hay un centro de control único.

Para los compradores, cuando se abre la venta de un evento popular, el sistema se cae o venden más entradas de las que caben. La sobreventa es un problema real.

Y para los que validan accesos en puerta, no hay forma rápida de saber si una entrada es auténtica o si ya la usaron antes.

VeriTix resuelve todo esto con una sola plataforma: centralizamos la operación, controlamos el stock con transacciones atómicas para evitar sobreventas, generamos tickets con hash único para eliminar falsificaciones, y automatizamos los cobros con Stripe."
